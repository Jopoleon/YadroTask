Необходимо разработать RESTful веб сервис, который реализует API для получения списка сетевых устройств на операционной системе Linux, а также подробную информация о каждом из них. Сервис должен корректно обрабатывать негативные сценарии и выдавать информацию об ошибках.
Необходимо выполнить тестирование разработанных компонент в отдельности и настроить непрерывное автоматическое тестирование.
Сервис должен устанавливаться на целевую систему в docker контейнере.
Исходный код проекта необходимо опубликовать в личном репозитории на https://github.com/, так же сервис должен быть доступен для установку из https://hub.docker.com.
